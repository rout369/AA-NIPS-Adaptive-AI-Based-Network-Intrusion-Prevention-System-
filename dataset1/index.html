NSL-KDD Dataset

A refined benchmark dataset for network intrusion detection systems, addressing critical limitations of the original KDD'99 dataset.

Version 1.0 125,973 records

Abstract

NSL-KDD is a refined dataset designed to overcome inherent limitations of the KDD'99 dataset. While acknowledging McHugh's critical analysis [2], this dataset serves as an effective benchmark for network-based intrusion detection systems research. With reasonable record counts in both training and testing sets, it enables comprehensive experiments without random sampling, ensuring consistent and comparable evaluation results across different research works.

Key Improvements

  • No Redundant Records

    Eliminates duplicate records in training set, preventing classifier bias towards frequent records.

  • Unbiased Test Sets

    No duplicate records in test sets, ensuring fair performance evaluation.

  • Balanced Difficulty Levels

    Inversely proportional selection from difficulty groups for better evaluation granularity.

  • Affordable Size

    Reasonable record counts enable full dataset experiments without random sampling.

Statistical Analysis

The original KDD dataset suffered from massive redundancy (4.9M records → 1.07M distinct), causing learning algorithms to bias toward frequent records. This issue particularly affected U2R and R2L attacks - the most harmful network threats. Our analysis reveals the extent of this redundancy problem:

Methodology

Three random 50K-record subsets were used to train 7 different learners (21 total models). Each record received a #successfulPrediction score (0-21) based on correct classifications. This rigorous annotation process ensures reliable difficulty assessment for benchmarking.
